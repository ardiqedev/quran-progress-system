# QEDEV Confirm Component

## Public API

```javascript
Confirm.init();

Confirm.show({
  title: "Delete Data",

  message: "Are you sure?",

  onConfirm() {
    console.log("Deleted");
  },
});
```

## Features

- Overlay
- Confirm Dialog
- Callback
- Responsive
- ESC Ready (V2)

Version: QEDEV Framework V1
