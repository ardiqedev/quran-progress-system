# QEDEV Empty Component

## Public API

```javascript
Empty.render();
```

Custom:

```javascript
Empty.render({
  icon: "📖",

  title: "Belum Ada Santri",

  message: "Silakan tambah data santri terlebih dahulu.",
});
```

## Features

- Reusable
- Dynamic Icon
- Dynamic Title
- Dynamic Message

Version: QEDEV Framework V1
