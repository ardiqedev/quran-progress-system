# Pagination Blueprint

## Purpose

Menampilkan navigasi halaman.

---

## Public API

```javascript
Pagination.render(options);
```

---

## Options

```javascript
{

    page:1,

    totalPage:10

}
```

---

## Features

- Previous Button
- Next Button
- Active Page
- Disabled State
- Responsive

---

## Responsibility

- Render Pagination
- Render Active Page

Tidak memiliki business logic.

---

## Version

QEDEV Framework V1
