# QEDEV Button Component

## Public API

```javascript
Button.render({
  text: "Simpan",

  type: "primary",
});
```

Dengan Icon

```javascript
Button.render({
  text: "Tambah",

  type: "success",

  icon: "➕",
});
```

Custom Attribute

```javascript
Button.render({
  text: "Delete",

  type: "danger",

  attributes: 'data-id="A001"',
});
```

## Features

- Multiple Style
- Icon Support
- Custom Attribute
- Disabled Support
- Reusable

Version: QEDEV Framework V1
