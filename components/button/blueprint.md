# Button Blueprint

## Purpose

Membuat tombol yang konsisten di seluruh QEDEV Framework.

## Public API

```javascript
Button.render(options);
```

## Types

- primary
- success
- warning
- danger
- info
- outline

## Responsibility

- Render button
- Mendukung icon
- Mendukung custom attribute
- Tidak memiliki business logic

Version: QEDEV Framework V1
