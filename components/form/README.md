# QEDEV Form Component

## Public API

```javascript
Form.input({
  label: "Nama",

  name: "nama",
});
```

```javascript
Form.select({
  label: "Kelas",

  name: "kelas",

  options: [
    {
      value: "A",
      label: "Kelas A",
    },
    {
      value: "B",
      label: "Kelas B",
    },
  ],
});
```

## Features

- Factory Pattern
- Reusable
- Lightweight
- Framework Ready

Version: QEDEV Framework V1
