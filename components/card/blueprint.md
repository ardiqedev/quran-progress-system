# Card Blueprint

## Purpose

Menampilkan kontainer informasi yang konsisten di seluruh QEDEV Framework.

## Public API

```javascript
Card.render(options);
```

## Structure

- Header (Optional)
- Body
- Footer (Optional)

## Responsibility

- Render Card
- Render Header
- Render Body
- Render Footer

Tidak memiliki business logic.

Version: QEDEV Framework V1
