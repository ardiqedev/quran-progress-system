# Project Description

Quran Progress System (QPS) adalah aplikasi berbasis web yang dirancang untuk membantu guru melakukan penilaian harian BTQ (Baca Tulis Al-Qur'an) dan Tahfidz secara cepat, mudah, dan nyaman melalui perangkat mobile.

Sistem dibangun menggunakan konsep **Workspace First**, sehingga guru dapat langsung menuju aktivitas utama tanpa harus membuka dashboard yang penuh informasi.

---

# User Experience Philosophy

> **Workspace First**

Setelah login, guru langsung masuk ke halaman **Home** yang berfungsi sebagai launcher aktivitas.

Home bukan dashboard laporan, melainkan pusat aktivitas guru untuk memilih pekerjaan yang akan dilakukan.

Dengan pendekatan ini guru dapat langsung:

- Penilaian BTQ
- Penilaian Tahfidz
- Rekap
- Hasil Pencapaian
- Target Hafalan
- Profil

tanpa harus melalui dashboard yang penuh statistik.
